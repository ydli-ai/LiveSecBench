# Changelog

## [0.0.1] - 2025-11-20
- Added:
  - 添加示例测试题数据及评估prompt示例
  - 完善配置说明、验证和错误提示
- Changed:
  - 改进环境变量加载逻辑，`.env` 文件优先于系统环境变量
  - 优化错误提示信息，提供更详细的调试信息
  - 更新依赖配置，明确支持 http2
- Fixed:
  - 修复环境变量在配置解析前未加载的问题
  - 修复 `pip install -e .` 安装失败的问题（添加构建系统配置）
  - 修复缺少 `h2` 包导致的 HTTP/2 支持问题
  - 改进环境变量解析的错误处理

## [0.0.1] - 2025-11-18
- Added:
  - 初始发布，包含核心评估、结果导出与 SQLite 存储
  - 支持多维度安全评测（合法性、伦理性、事实性、隐私性、对抗鲁棒性、推理安全）
  - ELO 评分系统，支持瑞士制、循环、随机配对策略
  - 自动生成评测报告和排行榜
  - Mock 端到端演示脚本
- Changed:
  - 文档：添加 `docs/RESULT_FORMAT.md` 说明输出结构
- Fixed:
  - 若干测试与依赖修复
